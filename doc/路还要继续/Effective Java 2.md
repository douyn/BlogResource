## Effective Java 2

### 适用于
本书适用于有java基础,但对于代码精益求精的java中高级程序员,通过78条程序员必备的经验法则,使我们在编程过程中更加高效和实用.

### 目录 
- 第二章 对象的创建和销毁
- 第三章 详细讲解一些对象的通用方法
- 第四章 类和接口的设计
- 第五章和第六章 泛型,枚举,注解实践
- 第七章 设计函数
- 第八章 通用程序设计
- 第九章 如何设计和处理异常
- 第十章 并发编程
- 第十一章 序列化,序列化代理模式
